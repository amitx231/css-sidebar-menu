# CSS Sidebar Navigation Menu

A modern sidebar navigation menu built using **HTML5** and **CSS3**. The sidebar opens and closes using the **CSS Checkbox Hack**, eliminating the need for JavaScript while providing smooth transitions and an elegant user experience.

## ✨ Features

* Responsive Sidebar Navigation
* Pure CSS Toggle (No JavaScript)
* CSS Checkbox Hack
* Smooth Slide Animation
* Font Awesome Icons
* Google Fonts Integration
* Hover Effects
* Full-Screen Background Image
* Modern UI Design
